El proyecto cuenta con 2 archivos .py y un .ipynb:

'main.py' es el archivo principal, automáticamente se ejecutará la aplicación Streamlit almacenada en el archivo 'PI_II.py', la ejecución del archivo 'main.py' tarda algunos segundos debido a que al ejecutarla se descarga el dataset nuevamente para actualizar los datos.

Tambien es posible ejecutar únicamente la aplicación al ejecutar el archivo 'PI_II.py' y se usarán los datos de 'COVID-19_Dataset.csv' que ya se encuentra descargado previamente.

Finalmente, el archivo 'PI_2.ipynb' es el jupyner notebook en donde se responde el cuestionario, la presentación y el dashboard se encuentran al ejecutar la aplicación 'PI_II.py'.
